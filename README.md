# Abdul's Portfolio 🚀

A stunning space-themed personal portfolio website showcasing my experience, projects, and skills.

## Features

- **Space Theme**: Beautiful dark space-themed design with animated stars and floating planets
- **Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Smooth Animations**: Engaging animations including glitch effects, floating objects, and smooth transitions
- **Modern Design**: Clean, professional layout with gradient accents
- **Fast Performance**: Optimized for quick loading and smooth scrolling
- **SEO Friendly**: Properly structured HTML with semantic elements

## Sections

- **Home**: Eye-catching hero section with call-to-action buttons
- **About**: Brief introduction and key statistics
- **Experience**: Professional experience and work history
- **Projects**: Showcase of featured projects with descriptions
- **Skills**: Technical skills with progress indicators
- **Contact**: Contact form and social media links

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Orbitron, Space Mono)

## Getting Started

1. Clone this repository:
```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

2. Open `index.html` in your browser to view the portfolio locally

3. Customize the content:
   - Edit `index.html` to update personal information
   - Modify `styles.css` for color schemes
   - Update project details and experience sections

## Customization Guide

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #00d4ff;      /* Cyan */
    --secondary-color: #ff006e;    /* Pink */
    --accent-color: #ffbe0b;       /* Yellow */
    /* ... more colors ... */
}
```

### Contact Information
Update the contact section in `index.html` with your actual details

### Projects
Modify the projects section to showcase your actual work with links

### Experience
Update your work experience timeline in the experience section

## GitHub Pages Deployment

### Option 1: Using `gh-pages` Branch
1. Create a new branch called `gh-pages`
2. Push all files to this branch
3. Go to your repository settings
4. Under "Pages", select `gh-pages` as the source
5. Your site will be available at `https://yourusername.github.io/portfolio`

### Option 2: Using `main` Branch with Docs Folder
1. Create a `docs` folder in your main branch
2. Move all files into the `docs` folder
3. Push to GitHub
4. Go to repository settings
5. Under "Pages", select `main` branch and `/docs` folder as source

### Option 3: Deploy Directly from Main
1. Ensure all files are in the root of your main branch
2. Go to repository settings
3. Under "Pages", select `main` branch as source
4. Your site will be live at `https://yourusername.github.io`
